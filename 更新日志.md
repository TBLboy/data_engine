# 更新日志

## 2026-07-06

### 修复
- **扫描入库填充时长/帧数**：MinIO 扫描时自动读取 manifest.json 提取 duration 和 frame_count，写入 Episode 和 EpisodeInventory，修复数据总库页面所有 episode 时长/帧数始终为 0 的问题。
- **定时扫描时区修正**：APScheduler CronTrigger 增加 timezone='Asia/Shanghai'，每日自动扫描从 UTC 0:00 改为北京时间 0:00 触发。
- **审核锁重复显示**：任务池页面审核锁列移除冗余纯文字，仅保留 el-tag。人工质检页面无人认领时不再重复显示。
- **人工质检锁面板样式修复**：scoped CSS 覆盖全局 flex 布局导致内容垂直堆叠、背景色不匹配。恢复 flex 居中布局及透明背景，新增 lock-actions 按钮对齐。

### 变更
- **任务类型列表对齐**：卡片横向占满宽度，文字左对齐，状态标签统一右对齐
- **任务类型名称字数限制**：创建和编辑对话框中名称输入框限制18字符，增加实时计数提示
- **仪表盘指标标签修正**：工作台"样本完成率"改为"抽检通过率"，小字说明同步更新

### 调研
- **LeRobot 生态对标研究**：完成 LeRobot 官方文档系统性学习，梳理数据集 v3.0 格式、子任务标注体系、lerobot-doctor 质检工具、Dataset Visualizer 可视化平台等
- **Dataset Visualizer 源码克隆**：已在本地克隆 visualize_dataset 项目（Apache 2.0），确认本地可部署
- **平台功能对比分析**：输出 L2/L3/L4 各层与 LeRobot 生态的差距报告
